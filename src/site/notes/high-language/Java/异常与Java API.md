异常类都继承自java.lang包下的（） 
A、Error类 
B、Exception类
C、Throwable类 
D、IOError类

在Java中，所有异常类都继承自`java.lang`包下的`Throwable`类。`Exception`类和`Error`类都是`Throwable`的子类。通常，`Exception`用于表示程序应该捕获的异常条件，而`Error`则用于指示严重的错误，通常不期望程序去捕获和处理这些错误
